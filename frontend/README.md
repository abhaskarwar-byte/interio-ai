# Interio AI 🏡✨

A luxury AI-powered interior design landing page built using React + Vite.

## 🚀 Features

- Premium dark luxury UI
- Gold-themed design system
- Smooth animations using Framer Motion
- Scrollable card-based layout
- Responsive hero section with background image
- Clean and modern component structure

## 🛠️ Tech Stack

- React (Vite)
- Tailwind CSS (optional styling)
- Framer Motion (animations)
- JavaScript (ES6+)
